Detroit: Android Legacy System
This is a C# project about Interfaces and Abstract Classes. I used the Detroit: Become Human game theme to show how OOP (Object-Oriented Programming) works.

What is inside?
Interfaces: I created IGreetable for basic actions like greeting.

Abstract Classes: I used AndroidTemplate and Deviant as base designs for all androids.

Inheritance: Classes like ServiceAndroid and WorkerAndroid get their features from the base classes.

How it works:
When you run the program, different androids introduce themselves. Depending on their version or type (Deviant or Standard), they give different reactions to the user.
